# Quantum × Finance

양자컴퓨팅의 금융 응용을 보여주는 인터랙티브 대시보드입니다.

## 배포 방법 (Vercel)

1. 이 폴더 전체를 GitHub repository에 업로드
2. https://vercel.com 에서 GitHub 계정으로 로그인
3. "New Project" → 이 repository 선택 → "Deploy" 클릭
4. 끝! 약 1~2분 후 공유 가능한 링크가 생성됩니다.

## 로컬에서 실행하려면

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속

## 사용 기술

- React 18
- Vite
- Tailwind CSS
- Recharts (차트)
- Lucide React (아이콘)
